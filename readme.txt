Add custom fields of post as meta tag in head section.

== Description ==

Add custom fields of any post as meta tag in head section of that post. It is useful for SEO purposes when you want to add any data as meta tags in head section of that post. You don't need any sophisticated plugin or learn coding to add meta tags for your keywords. Just add custom fields in your post and this plugin will automatically add those custom fields as meta tags in your post header.

== Installation ==

1. Upload `customfieldasmeta` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
